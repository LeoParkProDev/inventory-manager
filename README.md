# 인벤토리 매니저 Pro

실시간 재고 관리 웹 애플리케이션

## 소개

인벤토리 매니저 Pro는 Google Apps Script와 연동되어 실시간으로 재고를 관리할 수 있는 웹 기반 애플리케이션입니다. 직관적인 UI와 전체 동기화 모드를 통해 효율적인 재고 관리가 가능합니다.

## 주요 기능

- 실시간 재고 조회 및 업데이트
- 카테고리별 품목 분류
- 검색 및 필터링 기능
- 새로운 품목 추가
- Google Sheets와 자동 동기화
- 반응형 디자인 (모바일/태블릿/데스크톱)

## 기술 스택

- HTML5
- CSS (Tailwind CSS)
- Vanilla JavaScript
- Font Awesome Icons
- Google Apps Script API

## 사용 방법

1. 웹 브라우저에서 애플리케이션 접속
2. 자동으로 데이터 로드
3. 품목 추가 또는 수량 업데이트
4. 변경사항 자동 저장
